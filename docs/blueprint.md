# **App Name**: HydroTrack

## Core Features:

- Water Intake Logging: Log water intake with a single tap, incrementing the daily total.
- Daily Hydration Goal Setting: Set a personalized daily hydration goal in ounces or milliliters.
- Progress Visualization: View progress toward the daily goal via a progress bar or similar visual.
- Firestore Data Storage: Store user data, water logs, and daily summaries in Firestore.
- Daily Reset: Automatically reset the daily water intake total at midnight.
- User Authentication: Secure user accounts with email/password or social login, managed via Firebase Authentication.

## Style Guidelines:

- Primary color: Deep sky blue (#00BFFF) to represent water and hydration.
- Background color: Very light cyan (#E0FFFF), almost white, for a clean and fresh look.
- Accent color: Turquoise (#40E0D0) for progress indicators and interactive elements, slightly different hue from primary.
- Body and headline font: 'PT Sans', a humanist sans-serif for a balance of modernity and readability.
- Simple, water-themed icons (droplets, glasses) for intake logging and progress indicators.
- Clean, minimalist layout with a focus on the progress bar and single-tap logging button.
- Subtle animations (water droplet filling, progress bar increasing) to provide visual feedback.
